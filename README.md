# React-arco-admin

node 版本需要在v14之上

```bash
pnpm i 

pnpm start

```


## 该系统技术栈特点
- 基于webpack5.x配置
- eslint严格代码规范
- prettier统一代码风格
- husky代码提交检测
- 环境变量配置规范化

## 技术框架
- [x] react18
- [x] react-dom
- [x] react-router-dom 6.x
- [x] react-hook-form
- [x] hooks
- [x] arco-design

## 代码规范
- [x] eslint
- [x] prettier
- [x] husky
